# Next.js Sample

Minimal Next.js App Router sample project that renders "Hello World" at the root route.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 in your browser to view the sample.

## Build

Create a production build:

```bash
npm run build
```

Run the production server after building:

```bash
npm run start
```
